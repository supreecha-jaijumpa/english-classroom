# Translate Skill

Translate a word or short phrase and show an example sentence.

## Usage

```
/translate [word or phrase]
```

## Examples

```
/translate apple
/translate good morning
/translate I am tired
```

## Behavior

1. Translate the word or phrase to simple English (and show the native language meaning)
2. Show the pronunciation if it helps
3. Give 2 example sentences using the word
4. Add the word to `vocabulary/word-bank.md` if it is new
