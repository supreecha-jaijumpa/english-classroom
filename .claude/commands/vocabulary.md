# Vocabulary Skill

Show saved vocabulary words.

## Usage

```
/vocabulary
```

## Behavior

1. Read the most recent ~30 words in `vocabulary/word-bank.md` (newest are at the top — do not read the whole file once it grows large)
2. Read `vocabulary/phrases.md`
3. Display words grouped by topic or date learned
4. For each word show: the word, its meaning, and an example sentence
5. Highlight the 5 most recently added words
6. Mention the total word count from the counter at the top of the word bank
