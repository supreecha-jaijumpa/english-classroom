# Practice Skill

Start a free conversation practice session.

## Usage

```
/practice
```

## Behavior

1. Skip the structured lesson format
2. Start a casual, friendly conversation in English
3. Pick a topic based on the last 3 entries in `progress/log.md` (newest at the top) or ask what to talk about
4. Correct mistakes naturally — not after every sentence, only when important
5. Keep responses short so there is more back-and-forth
6. After 10+ exchanges, offer to run the `/wrap-up` checklist to save all progress
