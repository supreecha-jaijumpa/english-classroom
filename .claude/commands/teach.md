# Teach Skill

Jump directly to a specific English topic lesson.

## Usage

```
/teach [topic]
```

## Examples

```
/teach greetings
/teach numbers
/teach past tense
/teach shopping
```

## Behavior

1. Skip warm-up and review
2. Go directly to a focused lesson on the requested topic
3. Follow the teaching style from CLAUDE.md (simple English, patient, encouraging)
4. Include 3–5 practice exercises for the topic
5. At the end, add any new vocabulary to `vocabulary/word-bank.md`
6. Log the session in `progress/log.md`
