# Quiz Skill

Test knowledge from recent lessons.

## Usage

```
/quiz
```

## Behavior

1. Read `progress/log.md` to find the last 3 topics studied
2. Read `vocabulary/word-bank.md` for recent vocabulary
3. Ask 5 quiz questions mixing: vocabulary, grammar, and translation
4. Wait for answers one at a time
5. Mark each answer correct or gently correct mistakes with an explanation
6. Give a final score and encouragement at the end
