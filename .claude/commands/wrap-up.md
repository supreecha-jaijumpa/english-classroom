# Wrap Up Skill

End the current lesson and save all progress. This is the **single authoritative end-of-session checklist** — every file update happens here, and only here.

## Usage

```
/wrap-up
```

## Behavior

Work through this checklist in order:

1. **Correct** any remaining mistakes from this session
2. **Summarize**: what was practiced, what was learned
3. **`progress/log.md`** — add a new entry at the top with today's date and topic. If the log now has more than 20 sessions, move the oldest 10 to `progress/log-archive.md`
4. **`vocabulary/word-bank.md`** — add new words from this session at the top of the list and update the total counter (this is the only place the word count lives)
5. **`corrections.md`** — add repeated mistakes to "Active Mistakes"; if a recorded mistake was used correctly, increase its counter — at 3/3 move it to the "✅ Mastered" section
6. **`progress/streaks.md`** — update total sessions, current streak, best streak, the calendar mark, and tick any milestones reached
7. **`plan/this-week.md`** — tick today's daily task and add a short note if useful
8. **`grammar/notes.md`** — if a grammar topic was taught, add a short plain-language note
9. **`plan/roadmap.md`** — tick the topic checkbox if it is now finished; update "Current Stage" if a stage is complete
10. **`me.md`** — update "What I Find Difficult" / "What I Am Good At" if something new was noticed this session
11. End with one encouraging sentence

## Notes

- Only update files that actually changed this session — but always do steps 3 and 6.
- Other commands (`/study`, `/teach`, `/practice`) should point here instead of keeping their own save steps.
