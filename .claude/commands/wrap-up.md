# Wrap Up Skill

End the current lesson and save all progress.

## Usage

```
/wrap-up
```

## Behavior

1. Correct any remaining mistakes from this session
2. Give a short summary: what was practiced, what was learned
3. Append a new entry to `progress/log.md` with today's date and topic
4. Add any new vocabulary from this session to `vocabulary/word-bank.md`
5. Add any repeated mistakes to `corrections.md`
6. End with one encouraging sentence
