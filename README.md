# English Classroom

A personal English learning workspace powered by Claude AI — your patient mentor from A1 to C1.

## Goal

Daily conversation practice leading to advanced fluency.

## How to Start a Session

Open this folder in VS Code and talk to Claude:

| What to say | What happens |
|---|---|
| `let's study` | Full lesson session (warm-up → lesson → practice → wrap-up) |
| `/practice` | Quick conversation or exercise |
| `/quiz` | Vocabulary or grammar quiz |
| `/vocabulary` | Review your word bank |
| `/progress` | See your learning history |
| `/translate` | Translate a word or phrase |
| `/teach [topic]` | Learn something specific |

## Folder Structure

```
english/
├── CLAUDE.md            ← Teaching rules and session structure
├── me.md                ← Your English profile and goals
├── corrections.md       ← Your grammar mistakes and corrections
├── vocabulary/
│   ├── word-bank.md     ← New words learned each session
│   └── phrases.md       ← Useful phrases and expressions
├── grammar/
│   └── notes.md         ← Grammar rules you have studied
├── writing/
│   └── journal.md       ← Your writing practice
├── progress/
│   ├── log.md           ← Session-by-session history
│   ├── log-archive.md   ← Older session history
│   └── streaks.md       ← Study streak tracker
└── plan/
    ├── roadmap.md       ← Full A1 → C1 learning path
    └── this-week.md     ← Current week's focus
```

## Learning Path

```
A1 (Beginner) → A2 (Elementary) → B1 (Pre-Intermediate)
             → B2 (Upper Intermediate) → C1 (Advanced)
```

See [plan/roadmap.md](plan/roadmap.md) for detailed topics and milestone tests at each stage.

## Tips

- Study a little every day — 20–30 minutes is enough
- Don't worry about mistakes — Claude will correct gently
- Fill in [me.md](me.md) with your goals and preferences so lessons are tailored to you
