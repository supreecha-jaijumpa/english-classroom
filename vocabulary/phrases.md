# Daily Phrases

Useful phrases for real conversations. Organized by situation.

---

## Meeting People
| Phrase | When to use |
|--------|------------|
| "Hi! My name is ___." | Introducing yourself |
| "Nice to meet you!" | When meeting someone new |
| "How are you?" | Asking how someone is |
| "I'm fine, thank you." | Replying when someone asks |
| "Where are you from?" | Asking someone's country |
| "I'm from ___." | Saying your country |

## Everyday Situations
| Phrase | When to use |
|--------|------------|
| "Excuse me." | Getting someone's attention |
| "Can you help me?" | Asking for help |
| "I don't understand." | When you don't understand |
| "Can you repeat that?" | When you want them to say it again |
| "Can you speak slowly?" | When they speak too fast |
| "How much is this?" | Asking the price |

## Polite Expressions
| Phrase | When to use |
|--------|------------|
| "Please." | Making a polite request |
| "Thank you very much." | Thanking someone |
| "You're welcome." | Replying to "thank you" |
| "I'm sorry." | Apologizing |
| "No problem!" | Saying it's okay |
