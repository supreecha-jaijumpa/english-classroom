# My Writing Journal

Write a little bit in English every day or every week. Don't worry about mistakes — Claude will help you correct them!

## How to Write an Entry
Just write freely. Start with the date and a topic. Example:

> **June 11, 2026**
> Today I want to practice writing in English. My name is ___. I live in ___. I like ___.
