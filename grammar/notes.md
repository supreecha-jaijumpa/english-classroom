## Lesson 1: "There is" and "There are"

Use **"there is"** for ONE thing. Use **"there are"** for MORE THAN ONE thing.

| Amount | Rule | Example |
|--------|------|---------|
| One | there **is** a ... | "There is a game I like." |
| Many | there **are** many ... | "There are many tools to use." |

Tip: If the noun ends in **-s** (plural), use **"there are"**.
- "There is a **tool**." → one tool
- "There are many **tools**." → many tools
